Emojic
======

Emoji card game

Set up
------

1. ```yarn```

2. ```cp client/.env.example client/.env```

Run
---

### Dev

1. ```yarn start-server```

2. In a new terminal: ```yarn start-client```

### Prod

1. ```./build.sh```

2. Run and serve ```build/index.js``` with Node, e.g. ```node build/index.js``` or with server software like Nginx
